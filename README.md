Backend

    - email and password

      - activate account (30 min expiry) and first log in

      - reset password (15 min expiry)

    - OAuth 2.0 (Google)
  - addresses
    - list all addresses
    - add a new address
    - update an address
    - delete an address
  - profile
    - get profile
    - update profile
      - nickname
      - avatar url
